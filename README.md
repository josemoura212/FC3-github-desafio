# Desafio: Assinatura de Commits com GPG

## Objetivo

Configurar assinatura GPG no Git/GitHub para garantir autenticidade de commits.

## Requisitos

- Repositório público no GitHub
- Chave GPG configurada na máquina
- Chave GPG vinculada à conta do GitHub
- Commit assinado na branch `main`
- Commit exibindo badge **Verified** no GitHub

## Entregável

- Link do repositório público: _adicione aqui_

## Evidência

Este repositório contém commit assinado digitalmente com GPG.
Ao abrir o histórico de commits no GitHub, o commit deve aparecer com a badge verde **Verified**.
